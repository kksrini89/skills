# kksrini89/skills

> A collection of reusable **Agent Skills** for AI coding agents (Cursor, Claude, AmpCode, Continue.dev, etc.) built with the [Vercel Agent Skills](https://skills.sh) ecosystem.

These skills help boost developer productivity with focused, battle-tested instructions that agents can invoke on demand.

## 📂 Skills by Category

### 🛠️ [Engineering](./skills/engineering)
Skills focused on core software engineering practices, architecture, code quality, and automation.

### ⚡ [Productivity](./skills/productivity)
Skills that accelerate daily workflows, documentation, task management, and release processes.

## 🚀 Quick Start

Install any skill using the CLI:

```bash
npx skills add https://github.com/kksrini89/skills --skill git-commit-pr-writer
```

Or browse individual skill folders for manual installation.
