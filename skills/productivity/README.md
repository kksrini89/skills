# Productivity Skills

This category contains skills that help developers **ship faster** and reduce friction in daily repetitive tasks, especially around git workflows, documentation, PR writing, task management, and communication.

## Available Skills

### [`git-commit-pr-writer`](./git-commit-pr-writer)

**Description**:  
Generates high-quality, conventional commit messages, reviewer-friendly PR titles, and structured PR descriptions from staged changes or branch diffs.

**Key Benefits**:
- Enforces `commitlint` / Conventional Commit standards
- Produces professional PR descriptions with **Business Value**, **Changes**, **Validation**, and **Out of Scope** sections
- Saves significant time during releases and code reviews
- Automatically detects JIRA keys when present
- Safe — never runs destructive git commands

**Installation**:
```bash
npx skills add kksrini89/skills --skill git-commit-pr-writer
```

**Usage Examples**:

- "Write a commit and PR for my current changes"
- "Generate PR description using git-commit-pr-writer skill"
- "Create conventional commit for this feature"
